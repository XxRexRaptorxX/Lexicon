# Dragon's Breath

<ItemImage id="minecraft:dragon_breath" />

The <ItemLink id="minecraft:dragon_breath" /> can be obtained by scooping up the [*Ender Dragon's*](../creatures/boss-ender_dragon.md) breath attack or 
the purple effect clouds caused by dragon fireballs in an <ItemLink id="minecraft:glass_bottle" />. 

The [<ItemLink id="minecraft:dragon_breath" />](./dragons_breath.md) is a [*brewing*](../brewing.md) item that is used to make [*Lingering Potions*](../brewing/potion_types.md#lingering_potion).
