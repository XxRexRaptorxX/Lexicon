# Auto Lava/Water Farm

This simple farm uses the fact that a <ItemLink id="minecraft:pointed_dripstone" /> drain liquids that are above them downwards, 
so that they can be slowly collected with a <ItemLink id="minecraft:cauldron" />. 

This does not use up the liquids above them.
