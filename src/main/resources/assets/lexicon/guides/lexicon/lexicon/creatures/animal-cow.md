# Animal - Cow

__Health:__ 10     
__Attack Damage:__ 0    
__Behavior:__ Passive     
__Spawn:__ Overworld on grass, shearing [Mooshrooms](./animal-mooshroom.md)      
__Breeding:__ <ItemLink id="minecraft:wheat"/>     
__Properties:__     
An adult Cow can be milked by using a <ItemLink id="minecraft:bucket"/>. The *Cow* variant depends on the biome in which they spawn.
