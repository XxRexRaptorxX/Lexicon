# Redstone

Redstone is the *Minecraft* equivalent of electricity. 

Knowing the basics of redstone allows you to create amazing inventions. 

-----

You can obtain [*Redstone Dust*](./redstone/redstone_components.md#redstone) by mining <ItemLink id="minecraft:redstone_ore" /> underground. 

You can spread this dust across the ground as wire, attach it to a [<ItemLink id="minecraft:lever" />](./redstone/redstone_components.md#lever) or [*Doors*](./redstone/redstone_components.md#door), and craft it into 
components, like a [<ItemLink id="minecraft:redstone_torch" />](./redstone/redstone_components.md#torch) and a [<ItemLink id="minecraft:repeater" />](./redstone/redstone_components.md#repeater) to build huge machines.
