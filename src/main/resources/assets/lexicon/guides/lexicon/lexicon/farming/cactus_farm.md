# Auto Cactus Farm

This simple farm uses the property that <ItemLink id="minecraft:cactus" /> breaks and drops when it touches another block. 

The drops are then fed to a [<ItemLink id="minecraft:hopper" />](../redstone/redstone_components.md#hopper) using *Water*. 

The [<ItemLink id="minecraft:hopper" />](../redstone/redstone_components.md#hopper) then transports the items to a [<ItemLink id="minecraft:chest" />](../useables/chest.md).
