# Pottery Sherds

A *Pottery Sherd* is an item used to craft [<ItemLink id="minecraft:decorated_pot" />](../useables/decorated_pot.md)'s with ornamental designs. 

These sherds are mainly found in [<ItemLink id="minecraft:suspicious_sand" />](./suspicious_sand.md) or [<ItemLink id="minecraft:suspicious_gravel" />](./suspicious_sand.md#gravel) blocks of various 
[*Structures*](../world/structures.md) by searching them with a <ItemLink id="minecraft:brush" />.
